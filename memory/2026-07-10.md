## 明日方舟剧情 API 接入方法

### ArknightsStoryTextReader 数据源
- **Cloudflare R2 API**: `https://r2.m31ns.top/zh_CN/gamedata/excel/`
  - `story_review_table.json` → 剧情索引（含 storyTxt 路径）
  - 需要用户代理访问（服务器直连不上 Cloudflare R2）
- **GitHub 镜像（可直连）**: `https://raw.githubusercontent.com/Kengxxiao/ArknightsGameData/master/zh_CN/gamedata/story/[storyTxt].txt`
  - 服务器可以直接访问
  - 剧情原文是 .txt 格式，用 `grep '[name="角色名"]'` 提取对话
- **BWIKI 干员语音**: `https://wiki.biligame.com/arknights/index.php?title=[干员名]/默认/中文-普通话&action=raw`
  - `action=raw` 返回 wiki 源码（纯文本），服务器可访问
  - JS 渲染的页面(web_fetch)拿不到内容，必须用 raw 接口
  - 返回格式: `|语音类别=台词内容`

### 查台词流程
1. 从 `story_review_table.json` 找到 `storyTxt` 路径
2. 从 GitHub 镜像下载剧情 .txt 文件
3. 用 `grep '[name="角色名"]'` 提取对话
4. 或从 BWIKI 的 `action=raw` 页面获取干员语音
