# Skill & Tool 安装清单

> 此文件记录所有非云端自带的 skill 和 tool。
> 每次 session 重启后检查完整性。
> 恢复命令：bash install-skills.sh

## 安装状态检查

```bash
# 检查 skill 总数
ls ~/.openclaw/skills/ | wc -l

# 检查特定 skill 是否存在
for s in pocock-grill-me emil-apple-design taste-brandkit proxy-cn huashu-design llm-wiki iconify-skill browser-skill gufa-frontend; do
  [ -d ~/.openclaw/skills/$s ] && echo "✅ $s" || echo "❌ $s MISSING"
done
```

## 恢复流程

1. 运行 `bash install-skills.sh`（安装 Pocock + Emil + Taste = 35个）
2. 检查上述"单独安装的 Skills"是否存在
3. 缺失的用 git clone 补装
4. 检查 skillopt：`pip show skillopt`
