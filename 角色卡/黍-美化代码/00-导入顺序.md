# 黍 · 怀黍离 — 导入顺序

魅魔岛编辑页面，按以下顺序操作：

## 1. 开场白
在开场白**第一行**插入：
```
全局美化1
```
（这个会被正则捕获并替换为美化代码）

## 2. 正则脚本（共3个）

### 正则1：晴日风格
- 模式: `全局美化1`
- 替换: 粘贴 `黍-晴日风格.txt` 全部内容
- 标志: g
- 默认: 启用

### 正则2：雨夜风格
- 模式: `全局美化1`
- 替换: 粘贴 `黍-雨夜风格.txt` 全部内容
- 标志: g
- 默认: **禁用**（通过按钮切换启用）

### 正则3：侧边栏按钮
- 模式: `侧边栏按钮`
- 替换: 粘贴 `黍-侧边栏切换按钮.txt` 全部内容
- 标志: g
- 默认: 启用

## 3. 开场白完整示例
```
全局美化1
侧边栏按钮
（你的开场白正文...）
```

## 切换逻辑
- 点击「晴」→ 启用晴日风格，禁用雨夜
- 点击「雨」→ 启用雨夜风格，禁用晴日
- 点击「素」→ 禁用所有美化，恢复原样

## 注意
- 两个风格的正则模式都是 `全局美化1`，通过 disabled 属性切换
- 侧边栏按钮的 style id 必须与正则替换中的 style id 一致
- 晴日: `id="shu-style-sunny"`
- 雨夜: `id="shu-style-rain"`
