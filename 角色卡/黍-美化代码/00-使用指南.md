# 黍角色卡美化 — 使用指南

## 文件清单

| 文件 | 用途 | 放在哪里 |
|------|------|----------|
| 01-全局美化-日间.txt | 日间CSS主题 | 正则替换（触发词：全局美化1） |
| 02-全局美化-夜间.txt | 夜间CSS主题 | 正则替换（触发词：全局美化2） |
| 03-状态栏模板.html | 状态栏HTML模板 | 参考用，实际用06 |
| 04-正则脚本.json | 正则脚本配置 | 导入到魅魔岛正则 |
| 05-角色设定中的KV格式指令.txt | KV格式规范 | 粘贴到角色人设末尾 |
| 06-状态栏正则完整版.html | 状态栏正则替换 | 正则替换内容 |

## 导入步骤

### 第一步：在角色设定中加入KV格式指令
把 `05-角色设定中的KV格式指令.txt` 的内容粘贴到黍角色人设的**末尾**。

### 第二步：在开场白中加入触发标记
在黍的开场白**第一行**加入：
```
全局美化1全局美化2
```
（日间和夜间主题同时加载，用户通过侧边栏切换）

### 第三步：导入正则脚本
在魅魔岛编辑页面的「正则」区域，创建以下正则：

**正则1：全局美化-日间**
- 模式: `全局美化1`
- 替换: 内容来自 `01-全局美化-日间.txt`
- 标志: g

**正则2：全局美化-夜间**
- 模式: `全局美化2`
- 替换: 内容来自 `02-全局美化-夜间.txt`
- 标志: g
- 默认禁用（先显示日间）

**正则3：状态栏渲染**
- 模式: `LS▶([\\s\\S]*?)◀LE`
- 替换: 内容来自 `06-状态栏正则完整版.html`（注意将`$1`替换为正则捕获组）
- 标志: g

**正则4：选项按钮渲染**（可选）
- 模式: `DGS_KV_TIPS_START▶tips:(.*?);◀DGS_KV_TIPS_END`
- 替换: 选项按钮HTML
- 标志: g

### 第四步：测试
1. 保存角色卡
2. 开始新对话
3. 检查：
   - 全局美化是否生效（背景色、气泡样式）
   - 状态栏是否显示（场景、角色卡片、好感度条）
   - 后续消息中状态栏是否持续更新

## 设计风格

基于「怀黍离中国风PPT」和「墨迹笔刷素材」的视觉风格：
- 配色：哑光黑底 + 古铜金(#AC8B5F) + 暗红点缀(#B22222)
- 字体：Noto Serif SC（思源宋体）
- 纹理：墨迹笔刷质感，工业磨砂底纹
- 元素：中国风方块字标签（地/衣/行/心/志）
- 边框：无圆角，直线切割

## 自定义

### 修改配色
编辑 `:root` 中的 CSS 变量：
- `--primary`: 主色调（古铜金）
- `--accent`: 强调色（暗红）
- `--bg-main`: 背景色
- `--bubble-ai`: AI气泡颜色

### 添加角色头像
在角色设定中添加：
```
黍的c1img状态栏内容：https://你的图片链接
```

### 添加侧边栏按钮
后续可加入Tony侧边栏模块，提供快捷指令按钮。
