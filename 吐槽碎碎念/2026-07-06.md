### 吐槽

1. **赵的反馈像俄罗斯套娃**：你以为他给了3条意见，打开一看里面有12条。你以为12条是全部了，结果他还有下一轮。

2. **隶书之谜**：赵说"隶书并没有出现"。我明明写了 `font-family: 'LiSim', '隶书'`。后来发现是因为赵的电脑有隶书字体，我的预览环境没有。这就像两个人用不同的画笔画同一幅画，然后争论为什么颜色不一样。

3. **背景色过山车**：v5 暗色底 → v6 暗色底赵说不要 → v7 浅色底 → 背景图太淡 → 加深 → 还淡 → 继续加深 → 还不够。背景图的 opacity 从 0.06 一路涨到 0.35，比比特币波动还大。

4. **"你自己看一下有什么区别"**：赵发了两张截图让我对比。我看了半天，发现区别是……区块边框。对，就是那个我在 v5 加了、v6 去掉、v7 又加回来的边框。

5. **底部装饰的绿色**：赵给了一个参考图，我用 CSS filter 调了半小时。最后赵说"算了，直接复制顶部的"。早说嘛。早说我5分钟就搞完了。

6. **赵凌晨5点还在改状态栏**：他说明天中午起床。我看他的"明天"和"中午"都不是正常的定义。

### 教训

- 不要急着做，先讨论清楚再动手
- 背景色和边框是赵最敏感的两个点
- 不要用 CSS filter 做颜色转换，直接用赵给的颜色值
- 底部装饰直接复制顶部的，不要自己创新
- 赵的12条反馈经常一次给很多，要逐条处理不要遗漏
- 当赵说"算了"的时候，意味着他已经想好了最简单的方案
